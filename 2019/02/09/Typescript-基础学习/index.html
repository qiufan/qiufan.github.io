Typescript 基础学习

发表于 2019-02-09 | 分类于 基础学习

为了写个小程序，正好以后入职也是要接触Typescript，先稍稍学习下基础。

基础类型

变量 代码表示 例子 备注
布尔 boolean false,true 无
数字 number 0x,0b,0o 只有浮点，支持十，十六，二，八进制
字符串 string (用””或者’’包裹，也可以用’’包裹，${expr}嵌入)
数组 type[],Array
元组 [type1,type2] [string,name] 用数组的形式包括两个类型就是元组，可以越界访问，会自动用联合类型替代
枚举 enum enum Color {Red=1,Green,Blue} 可以手动赋予初始值
Any any any[]=[1,true,”free”] 可以视作任意，包括可以在上面调用函数
void undefined,null
null和undefined 默认情况它们是所有类型的子类型，–strictNullChecks可以规定只能赋给void和它们自己，同时使用联合类型
Never never function error(message: string): never 永远不存在，必须在函数无返回，或者报错
object 非原始类型 除了number,string,boolean,symbol,null和undefined之外的类型

类型断言的方式有两种,someValue或者someValue as string
